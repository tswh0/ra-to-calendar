# Privacy Policy

Last updated: 2026-03-19

## RA to Calendar

RA to Calendar is a browser extension that helps users export Resident Advisor event pages as calendar entries.

## What Data the Extension Uses

The extension processes content from Resident Advisor event pages on `https://ra.co/*`, including:

- event title
- event date and time
- venue name
- event address
- lineup information
- event URL
- event description when available

## How the Data Is Used

The extension uses this data only to provide its core functionality:

- generate a local `.ics` calendar file
- open a prefilled Google Calendar event link
- display event information in the extension popup

## Data Collection and Sharing

RA to Calendar:

- does not require a user account
- does not collect personal profile information
- does not use analytics or tracking
- does not send data to a developer-controlled server
- does not sell or transfer user data to third parties

## Data Storage

The extension does not maintain a remote database and does not store user data outside the user's browser workflow needed to generate the requested export.

## Remote Code

The extension does not use remote code. All logic required for the extension is packaged with the extension itself.
